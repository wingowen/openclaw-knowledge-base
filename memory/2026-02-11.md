## Tailscale 访问 OpenClaw Web UI 问题

**问题**: 用户想通过 Tailscale IP 访问 OpenClaw Web UI

**尝试方案**:
- 设置 `gateway.bind = "tailnet"` 
- 但 Gateway 仍然只绑定到 127.0.0.1:18789

**根本原因**:
- Tailscale 运行在 **Windows 主机** 上，而不是 WSL2 内部
- WSL2 内部没有 tailscale/ts 网络接口
- `gateway.bind = "tailnet"` 在 WSL2 环境无法生效

**解决方案思路**:
1. 使用 `gateway.bind = "lan"` 让 Gateway 监听所有接口
2. Windows 主机上用端口转发或 Tailscale Serve 代理到 WSL2
3. 或者直接在 Windows 上运行 `tailscale serve` 代理到 WSL2 的端口

**待确认**: 用户需要确认是在 Windows 主机还是 WSL2 内部访问
