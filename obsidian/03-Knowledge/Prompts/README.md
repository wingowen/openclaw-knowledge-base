## 🎯 使用指引

### 场景 1：生成 A 股复盘报告
1. 运行 `python3 scripts/generate_daily_report.py`
2. 参考 `A股复盘报告Prompt.md` 完善分析
3. 输出到 `04-Reviews/`

### 场景 2：分析一个新项目
1. 参考 `项目研究Prompt.md` 的分析维度
2. 使用 `gh` CLI 采集基础数据
3. 输出到 `02-Projects/`

### 场景 3：技术选型对比
1. 参考 `通用分析框架.md` 的对比矩阵
2. 按权重评分决策
3. 记录到相关项目文档
