# 日报生成与推送逻辑

> **用途**：自动化日报的数据采集 → 生成 → 推送流程
> **涉及脚本**：`generate_daily_report.py`、`check_trading_day.py`、`send_qq_mail.py`

---

## 整体流程

```
┌─────────────────────────────────────────────────────────┐
│  1. 交易日判断                                            │
│  python3 check_trading_day.py --quiet                    │
│  → 非交易日提前退出                                        │
└───────────────────────┬─────────────────────────────────┘
                        │ 是交易日
                        ▼
┌─────────────────────────────────────────────────────────┐
│  2. 数据采集                                              │
│  腾讯财经 API: https://qt.gtimg.cn/q={codes}             │
│  - 4 个指数：上证/深证/创业板/科创50                        │
│  - 10 只关键个股                                          │
│  解析字段：现价/涨跌幅/成交额/振幅/换手率                     │
└───────────────────────┬─────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────────┐
│  3. 报告生成                                              │
│  - 模板：obsidian/Templates/A股复盘模板.md                │
│  - 输出：obsidian/04-Reviews/YYYY-MM-DD-A股深度复盘...md  │
│  - 包含 8 章节完整分析                                     │
└───────────────────────┬─────────────────────────────────┘
                        │
                        ▼
┌─────────────────────────────────────────────────────────┐
│  4. 多渠道推送                                            │
│  ├─ Obsidian：写入 vault（自动）                          │
│  ├─ GitHub：git add/commit/push（自动）                   │
│  ├─ 飞书：Webhook 推送关键结论（可选）                     │
│  └─ QQ 邮件：HTML 邮件发送（可选）                         │
└─────────────────────────────────────────────────────────┘
```

## Cron 配置

```bash
# 每个交易日 15:30 运行（收盘后 30 分钟）
30 15 * * 1-5 /bin/bash /root/.openclaw/workspace/scripts/run_daily_report.sh
```

`run_daily_report.sh` 内容：
```bash
#!/bin/bash
export FEISHU_WEBHOOK_URL="..."
export QQ_MAIL_USER="..."
export QQ_MAIL_AUTH_CODE="..."
export QQ_MAIL_TO="..."

cd /root/.openclaw/workspace
python3 scripts/check_trading_day.py --quiet || exit 0
python3 scripts/generate_daily_report.py
```

## 错误处理

| 异常 | 处理 |
|------|------|
| 非交易日 | 静默退出，不生成报告 |
| API 超时 | 重试 3 次，间隔 5 秒 |
| 数据解析失败 | 记录日志，标记为 `-` |
| Git push 失败 | 本地保留，下次重试 |
| 邮件发送失败 | 记录日志，不阻塞流程 |
