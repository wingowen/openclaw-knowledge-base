# Obsidian Vault（OpenClaw Workspace）

这是工作区内的本地知识主仓（Obsidian Vault）。

## 目录约定

- `00-Inbox/`：临时想法、待分类内容
- `01-Daily/`：每日记录（日志/节奏）
- `02-Projects/`：项目文档
- `03-Knowledge/`：方法论、知识卡片
- `04-Reviews/`：复盘（市场/任务/周报）
- `05-Archive/`：归档
- `Templates/`：模板

## 工作流（建议）

1. 先写 Obsidian（本地草稿 + 双链）
2. 定稿后同步到 Notion（对外展示/协作）

> 已保持：默认生成文档继续同步 Notion。
