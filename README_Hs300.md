# 沪深300成分股量价策略回测

## 项目概述

对沪深300成分股进行量价策略回测，验证「放量买入」和「缩量后的放量」策略的有效性。

## 文件结构

```
.
├── README.md                    # 本文件
├── hs300_stocks.csv           # 沪深300成分股列表 (124只)
├── hs300_fetch.py             # 数据分批获取脚本
├── hs300_full_report.txt      # 全量回测报告
│
└── stock_data/hs300/          # 股票历史数据
    ├── 000002.csv             # 万科A
    ├── 000333.csv             # 美的集团
    ├── 600519.csv             # 贵州茅台
    └── ... (共119只股票)
```

## 回测结果

### 策略1: 放量买入
- **总信号**: 5817笔
- **胜率**: 57.7% ✅

### 策略2: 缩量后的放量
- **总信号**: 5817笔
- **胜率**: 51.5%

## 核心结论

| 策略 | 胜率 | 评价 |
|:---|:---:|:---|
| 放量买入 | **57.7%** | ✅ 有效 |
| 缩量后放量 | 51.5% | ⚠️ 勉强有效 |

## 使用方法


## 结论

**放量买入策略整体有效**，在周期股（铜/金/铝）和消费股（白酒/醋）上效果最好，但在银行股上效果较差。
