# 📊 Reports - 报告归档

所有发送给用户的报告自动保存在此目录，并推送至 GitHub。

## 目录结构

| 目录 | 用途 | 示例 |
|------|------|------|
| `daily-review/` | 每日 A 股复盘与预判 | 2026-03-17-A股复盘.md |
| `stock-analysis/` | 个股/板块深度分析 | 比亚迪技术分析.md |
| `strategy/` | 周度/月度策略报告 | 周度策略报告-0317.md |
| `other/` | 其他类型报告 | 备用 |

## 规则

1. **所有报告必须归档**：发送给用户的报告同步保存到此目录
2. **只保留最终版**：多次修改的文件覆盖保存，不保留中间版本
3. **自动推送**：报告保存后推送到 GitHub
4. **命名规范**：`YYYY-MM-DD-报告标题.md`
