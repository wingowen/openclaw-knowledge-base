## ⚠️ 注意事项

1. **如果无法登录:**
   - 检查 Supabase 后端是否运行
   - 确认管理员账户已创建
   - 查看浏览器控制台错误信息

2. **如果 API 请求失败:**
   - 检查网络请求 (F12 → Network 标签)
   - 确认 Supabase URL 和密钥配置正确
   - 查看服务器日志

3. **浏览器控制台检查:**
   - 按 F12 打开开发者工具
   - 查看 Console 标签是否有错误
   - 查看 Network 标签检查 API 请求
