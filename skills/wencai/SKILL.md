# 问财股票筛选

通过浏览器访问同花顺问财，使用自然语言筛选股票。

## 快速使用

### 1. 构造搜索URL

```
https://www.iwencai.com/unifiedwap/result?w=<URL编码的条件>&querytype=stock
```

### 2. 浏览器操作流程

```javascript
// 打开问财搜索结果页
browser.open(targetUrl: "https://www.iwencai.com/unifiedwap/result?w=...&querytype=stock")

// 等待加载后获取快照
browser.snapshot()

// 解析结果表格获取股票列表
```

## 常用条件示例

**技术形态：**
- 近10日有放量上涨
- 最近3日成交量明显萎缩
- 股价在20日均线上方
- MACD金叉
- 突破前高

**基本面：**
- 市盈率小于20
- 净利润增长率大于30%
- ROE大于15%

**排除条件：**
- 非ST股
- 非创业板非科创板非北交所

**组合示例：**
```
近10日有放量上涨，最近3日成交量明显萎缩，股价在20日均线上方，非ST股，非创业板非科创板非北交所
```

## 结果解析

搜索结果页包含：
- 已选条件列表
- 筛选出的股票数量（如"选出A股 165"）
- 股票列表表格：序号、代码、名称、现价、涨跌幅、技术指标等

## 注意事项

- 条件之间用逗号或空格分隔
- 复杂条件建议分步筛选
- 问财会自动解析自然语言条件
